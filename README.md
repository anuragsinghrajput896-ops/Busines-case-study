# 📊 Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer churn behavior using SQL and Power BI. The goal is to identify key factors that lead to customer churn and provide actionable business insights.

---

## 🛠️ Tools & Technologies Used
- MySQL (SQL)
- Power BI
- Excel / CSV

---

## 📂 Project Files
- `ChurnAnalysis.sql` → SQL queries and database setup  
- `ChurnDashboard.pbix` → Interactive Power BI dashboard  
- `Customer_Churn_Analysis_Report.pdf` → Business insights and analysis  

---

## 📊 Key Analysis Performed
- Active vs Churned Customers Analysis  
- Subscription Type vs Churn Rate  
- Customer Engagement Analysis  
- Feedback Score Analysis  
- Inactive Users Identification  
- Monthly Churn Trend  
- Age Group Wise Churn

---

## 🔍 Key Insights
- Monthly subscription users show higher churn compared to yearly plans  
- Low engagement and low feedback customers are more likely to churn  
- Inactive users have the highest churn probability  
- Higher engagement leads to better customer retention  

---

## 💡 Business Recommendations
- Convert monthly users to yearly plans  
- Improve customer engagement strategies  
- Act on low feedback quickly  
- Re-engage inactive customers  

---

## 🚀 Conclusion
This project demonstrates how data analysis using SQL and Power BI can help businesses understand customer behavior and reduce churn through data-driven decisions.
